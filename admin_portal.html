⚠️ Danger Zone
These actions are irreversible. A backup is automatically taken before any wipe operation and emailed to Vishnu.Prasanth@infinite.com before deletion begins.
🗑️ Wipe All Table Data
Removes ALL data from: users, employees, tickets, workflow_states, audit_logs, allocation_records.
Table structures are preserved. A full backup is downloaded to your machine and emailed before the wipe begins.
After wipe, only the IntelliOnboard Admin account is re-seeded automatically.
👥 Wipe Employee Data Only
Removes all rows from employees and allocation_records tables only.
Users, tickets, and workflow history are preserved.
Use this before uploading a fresh allocation file.
🎫 Wipe Ticket Data Only
Removes all rows from tickets and workflow_states tables only.
Employees and user accounts are preserved.
